# AI Training with Jac

## Quick Start

1. Activate Python 3.11 environment:
   ```powershell
   .\.venv311\Scripts\Activate.ps1
   ```

2. Run the sample program:
   ```powershell
   jac run myfile.jac
   ```

## About Jac

Jac is a modern AI programming language with nodes, walkers, and built-in LLM support.
